css的简介

什么是css

层叠样式表，css是对html进行样式修饰的语言

层叠：就是层层的覆盖叠加，如果不同的css样式对同一html进行修饰，样式有冲突的部分应用优先级高的，不冲突的部分共同作用样式表，就是css属性样式的集合

样式表：就是css属性样式的集合

css作用

  1. 修饰html，使其html样式更加好看
  2. 提高样式代码的复用性
  3. html的内容与样式相分离，便于后期维护

css的引入方式和书写规范

  1. 内嵌样式

    内嵌样式是把css的代码嵌入到html标签中

    语法：

    1. 使用style属性将样式嵌入到html标签中
    2. 属性的写法：属性：属性值
    3. 多个属性之间使用分号隔开

  2. 内部样式

    在head标签中使用style标签进行css的引入

    语法：

    1. 使用style标签进行css的引入
      1. 属性：type：告知浏览器使用css解析器去解析
    2. 属性的写法：属性：属性值
    3. 多个属性之间使用分号隔开

  3. 外部样式

    将css样式抽取成一个单独的css文件，谁去使用给谁

    语法：

    1. 创建css 文件，将css属性写在css文件中
    2. 在head中使用link标签进行引入
      1. rel代表要引入文件与html的关系
      2. type：告知浏览器使用css解析器去解析
      3. href：css文件地址
    3. 属性的写法：属性：属性值
    4. 多个属性之间使用分号隔开

link与@import区别

  1. link所有的浏览器都支持，import部分低版本的IE不支持
  2. import方式是等待html加载之后再加载
  3. import方式不支持js的动态修改
