HTML基本标签

文件标签

  • html :根标签
  • head
  • body

排版标签

  • 注释标签
  • 换行标签
  • 段落标签

    • 属性 ：align:对齐方式 left canter right
  • 水平线标签
    • 属性：width:长度 50% size：粗细 10px color:颜色 align：对齐方式

块标签

  • div 行级块标签
  • span 行内块标签

文字标签

  • 基本文字标签 :
    • color 颜色
    • size 大小
      • 最大值：7
      • 最小值：1
      • 默认值：3
    • face 字体类型 黑体、宋体…
  • 标题标签
    • h1到h6
      • 依次减小 默认字体加粗 默认占据一行

清单标签

  • 无序列表

      • 属性：type：disc square circle

      • :列表项

    • 有序列表：

        • type: 1, A,I, a,i
        • 列表项

      图形标签

        • src:图形的地址

        • width：宽度

        • heigh：高度
        • border：边框
        • align：对齐方式 代表图片与相邻的文本的相对位置 top midde bottom
        • alt ：图片的文字说明

      链接标签

      • <a>
        • href 跳转的页面地址
        • name 名称
        • target：_selt _blank
